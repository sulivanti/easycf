## 2. UX-MCP-001 — Gestao de Agentes e Acoes

### 2.1 Jornada Principal (Happy Path)

```
1. Admin acessa /automacao/agentes
2. GET /admin/mcp-agents carrega lista de agentes (painel "Agentes" ativo por padrao)
3. Admin cria novo agente via drawer (nome, codigo, owner, scopes)
4. POST /admin/mcp-agents retorna 201 com api_key
5. Modal ApiKeyModal exibe key UMA VEZ — admin deve copiar e confirmar checkbox
6. Agente aparece na tabela com status ACTIVE
7. Admin navega para painel "Catalogo de Acoes" — cria/edita acoes
8. Admin navega para painel "Permissoes" — vincula acoes a agentes via checkboxes
```

### 2.2 Estados da Tela

| Estado | Comportamento | Componente |
|---|---|---|
| **loading** | Skeleton de tabela com 5 linhas placeholder | Skeleton rows |
| **loaded** | Tabela populada com agentes/acoes/matriz conforme painel ativo | AgentsTable / ActionsCatalog / PermissionsMatrix |
| **empty** | Ilustracao + "Nenhum agente cadastrado. Crie o primeiro agente MCP." | EmptyState com botao "Criar Agente" |
| **error** | Toast com mensagem RFC 9457 + correlationId copiavel | Toast de erro |
| **saving** | Botao de submit desabilitado + spinner inline | Button loading state |

### 2.5 Alternativas e Erros

| Situacao | Comportamento UX |
|---|---|
| Criar agente com escopo bloqueado | Inline error no campo scopes: "Escopo bloqueado para agentes MCP: {scope} (corresponde ao padrao {pattern})" (422) |
| Criar agente com codigo duplicado | Inline error: "Ja existe um agente com este codigo neste tenant." (409) |
| Revogar agente — confirmacao | Modal: "Revogar o agente '{nome}'? Esta acao e irreversivel." + campo obrigatorio "Motivo da revogacao" |
| Revogar agente REVOKED | Botao "Revogar" desabilitado (badge "REVOGADO") |
| Rotacionar key — confirmacao | Modal aviso amarelo: "A chave atual sera invalidada imediatamente. O agente ficara inoperante ate a nova chave ser configurada. Continuar?" |
| ApiKeyModal — tentar fechar sem checkbox | Botao "Fechar" desabilitado. Tooltip: "Confirme que copiou a chave antes de fechar." |
| ApiKeyModal — navegacao acidental | `beforeunload` warning: "A chave nao sera exibida novamente. Voce copiou a chave?" |
| Editar agente REVOKED | Formulario desabilitado. Banner: "Agente revogado — somente leitura." |
| Admin sem scope mcp:agent:revoke | Botao "Revogar" oculto (guard de scope no manifest) |
| Admin sem scope mcp:action:write | Painel "Catalogo de Acoes" somente-leitura (botao "Criar Acao" oculto) |
| Erro de rede generico | Toast: "Erro de conexao. Verifique sua rede e tente novamente." + correlationId |

## 3. UX-MCP-002 — Monitor de Execucoes MCP

```
1. Admin acessa /automacao/execucoes
2. GET /admin/mcp-executions carrega lista com metricas 24h no header
```
