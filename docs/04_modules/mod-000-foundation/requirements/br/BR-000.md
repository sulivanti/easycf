## BR-000 — Governança de Documentos Normativos para Geração Automática de Código

- Regra: O framework de geração automática de código deve reconhecer, aplicar e validar os documentos normativos (DOC-DEV-001, DOC-GNP-00) como "lei do projeto".
- Exemplo: Ao rodar `scaffold-module`, toda estrutura e metadado deve ser construída validando a User Story.
- Exceções: Nenhuma prevista que fuja às regras de ADR documentadas.
- Impacto: Padronização governada, evitando boilerplate inconsistente.
