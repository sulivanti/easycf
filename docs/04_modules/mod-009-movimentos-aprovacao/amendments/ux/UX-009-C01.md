### Mapeamento hex → token semântico

| Hex hardcoded | Token Tailwind | Contexto |
|---|---|---|
| `text-[#111111]` | `text-a1-text-primary` | Texto principal (headers, valores, body) |
| `text-[#888888]` | `text-a1-text-auxiliary` | Texto secundário (solicitante, data, caption) |
| `text-[#2E86C1]` | `text-primary-600` | Links e destaques primários |
| `text-[#CCCCCC]` | `text-a1-text-placeholder` | Placeholder de inputs |
| `border-[#E8E8E6]` | `border-a1-border` | Bordas de tabela, cards, inputs |
| `bg-[#F5F5F3]` | `bg-neutral-50` | Background de header de tabela e linhas expandidas |
| `text-[#16a34a]` | `text-success-500` | Texto do botão Aprovar |
| `border-[#16a34a]` | `border-success-500` | Borda do botão Aprovar |
| `bg-[#16a34a]` | `bg-success-500` | Background do botão Confirmar Aprovação |
| `hover:bg-[#d1fae5]` | `hover:bg-success-50` | Hover do botão Aprovar (outline) |
| `hover:bg-[#15803d]` | `hover:bg-success-600` | Hover do botão Confirmar Aprovação |
| `text-[#dc2626]` | `text-danger-500` | Texto do botão Rejeitar |

| `placeholder:text-[#CCCCCC]` | `placeholder:text-a1-text-placeholder` | Placeholder do textarea |
| `hover:text-[#111111]` | `hover:text-a1-text-primary` | Hover do botão Cancelar |

### Regra

Todas as cores hex literais em classes Tailwind arbitrárias (`text-[#...]`, `bg-[#...]`, `border-[#...]`) DEVEM ser substituídas por tokens semânticos do design system. Após aplicação, `grep '#[0-9A-Fa-f]' MovementsPage.tsx` deve retornar 0 resultados.
