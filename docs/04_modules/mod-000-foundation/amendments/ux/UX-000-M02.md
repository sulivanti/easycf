### D2 — Componente MetricCard (novo)

Criar componente `dashboard/MetricCard`:

```
MetricCard (269xauto, fill #FFFFFF, r:16, border 1px #E8E8E6, p:24px)
  Label (10px 700 uppercase ls:+1px #888888, mb:8px)
  Valor (36px 800 lh:40px, cor semantica ou #111111)
  Indicador (mt:10px, flex align-center gap:6px)
    Dot (8x8, r:50%, cor semantica)
    Texto (11px 400 #AAAAAA)
```

Props:
- `label: string` — texto uppercase (ex: "PROCESSOS ATIVOS")
- `value: string` — valor grande (ex: "12")
- `valueColor?: string` — cor do valor (default #111111)
- `dotColor: string` — cor do dot indicador
- `indicator: string` — texto do indicador (ex: "Em execucao")
