## Detalhamento

A OrgTreePage DEVE adotar layout split-panel em vez de lista flat:

```
ContentArea (1200x836, fill #F5F5F3)
  PainelArvore (380px, fill #FFFFFF, border-right 1px #E8E8E6)
    Titulo + Descricao + SearchBar + Tree hierarquica
  PainelDetalhe (flex, fill #F5F5F3, padding 24px, overflow-y auto)
    HeaderDetalhe + CardDadosCadastrais + CardDepartamentos + MetricCards
```

- Tree panel: 380px fixo, fundo branco, borda direita
- Detail panel: flex restante, fundo cinza claro, scrollavel
- Search bar DENTRO do tree panel (340x40), nao no header

    BotaoVoltar (ArrowLeft 20x20) + Titulo (18px 700 #111) + BotaoFechar (X 20x20)

- Arvore e OCULTA quando form abre
