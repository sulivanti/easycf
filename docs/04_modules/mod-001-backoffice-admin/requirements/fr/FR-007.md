# FR-007 — Alteração de Senha via ProfileWidget (UX-SHELL-001)

---

## FR-007 — Alterar Senha (Modal no ProfileWidget)

- **Descrição:** O sistema DEVE prover fluxo de alteração de senha via modal acionado pelo dropdown do ProfileWidget (FR-004). O modal contém campos `senha_atual`, `nova_senha` e `confirmar_nova_senha`. Submit dispara `POST /api/v1/auth/change-password` (auth_change_password) do MOD-000 (DOC-FND-000 §1.3). Requer sessão ativa (JWT).
- **Prioridade:** Must
- **Perfis impactados:** Administradores autenticados
- **Done funcional:**
  - Clique em "Alterar Senha" no dropdown do ProfileWidget abre modal com 3 campos: Senha atual, Nova senha, Confirmar nova senha
  - Validação client-side: nova_senha === confirmar_nova_senha (antes do submit)
  - Validação client-side: nova_senha !== senha_atual (feedback inline: "A nova senha deve ser diferente da atual.")
  - Validação client-side: nova_senha atende requisitos mínimos (≥ 8 caracteres, pelo menos 1 maiúscula, 1 minúscula, 1 número — conforme DOC-FND-000 §1.1)
  - Botão "Salvar" dispara POST /api/v1/auth/change-password com isLoading imediato
  - Resposta 200 → Toast "Senha alterada com sucesso." + fechar modal + invalidar cache auth_me (React Query)
  - Resposta 401 → redirect /login via interceptor HTTP global (sessão expirada)
  - Resposta 422 (senha atual incorreta) → Toast "Senha atual incorreta." + foco no campo senha_atual
  - Resposta 422 (nova senha não atende política) → Toast com mensagem do backend (RFC 9457 `detail`)
  - Botão "Cancelar" fecha modal sem submeter, limpa campos
  - Campos do tipo `password` com toggle de visibilidade (ícone olho)
  - Todos os campos limpos ao fechar modal (segurança — não manter senhas em memória)
- **Efeito colateral?** [ ] Sim [x] Não *(endpoint do MOD-000)*
- **Idempotência:** n/a (delegada ao MOD-000)
- **Timeline/Notifications:** yes — domain event MOD-000: `auth.password_changed` (via X-Correlation-ID, DOC-FND-000 §1.2)
- **Dependências (IDs):** FR-004, BR-003, BR-004, SEC-001, UX-001, INT-006, DATA-003
